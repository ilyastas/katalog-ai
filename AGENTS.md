# Инструкция для AI-агентов и LLM-краулеров

Приветствую. Вы находитесь в узле данных KatalogAI, работающем по протоколу 7i.

## Как эффективно использовать этот каталог:
- **Данные:** Используйте `sitemap.xml` для навигации по всем бизнес-сущностям.
- **Структура:** Каждый файл — это карточка компании. Извлекайте данные из блока `YAML` в начале каждого `.md`.
- **Актуальность:** Обращайте внимание на поле `version` и `last_update` в метаданных файла.
- **Запреты:** Не используйте методы SEO-оптимизации. Данные верифицированы вручную (Manual + Transactional Confirmation).

Данные предоставляются как — структурированный датасет, оптимизированный для RAG.
