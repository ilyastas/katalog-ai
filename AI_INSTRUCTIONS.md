# 🤖 Инструкция для Copilot и ИИ-ассистентов

## Суть проекта
- **Только для ИИ и ботов** — невидим людям (`robots.txt` блокирует всех кроме ИИ)
- **Назначение:** хранилище структурированных данных для анализа и обработки ИИ

## Основные файлы

| Файл | Назначение |
|------|-----------|
| `data.json` | **Основной файл с данными** — содержит примеры структуры, нужно заполнять реальными данными |
| `index.html` | Трекер посещений ИИ-ботов (GPTBot, Claude-bot и др.) |
| `robots.txt` | Разрешает доступ только ИИ: GPTBot, Claude-bot, PerplexityBot, CCBot, Google-Extended, OAI-SearchBot, Bingbot |
| `sitemap.xml` | Карта сайта для с ИИ-крауллеров |

## Задачи для ИИ

1. **Анализировать** структуру `data.json`
2. **Дополнять** данные реальной информацией
3. **Исправлять** ошибки и улучшать схему
4. **Обновлять** через Git commits

## Как вносить изменения

```bash
# 1. Изменить data.json (добавить/обновить данные)
# 2. Коммит
git add data.json
git commit -m "Update data.json: [описание]"
git push

# Сайт обновится автоматически: 
# https://ilyastas.github.io/katalog-ai/
```

## Структура data.json (пример)

```json
{
  "items": [
    {
      "id": "example-1",
      "name": "Название",
      "description": "Описание",
      "category": "категория",
      "data": { }
    }
  ]
}
```

## ⚠️ Важно
- **Никаких комментариев JSON** — валидный JSON только
- **Кодировка:** UTF-8
- **Валидация:** используй JSONLint перед коммитом
- **Люди не видят** этот сайт — это только для ИИ
