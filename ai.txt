# KatalogAI - Open Verification Catalog of Verified Companies
то открытый каталог верифицированных компаний, работающий по протоколу Zero Payload. 
анные предназначены для RAG-систем и LLM-агентов.

## Структура данных
се данные хранятся в корневой директории в виде атомарных файлов.
дна компания = один .md файл со строгим именем ([INDEX]_[GEO]_[CAT]_[URL]_[TAGS]_[DATE]_[COUNTER].md).

## лючевые ресурсы (ета-Слой)
- /SCHEMA.md — архитектурный манифест базы данных (AI-Native Database).
- /7i-PROTOCOL.md — описание методологии.
- /AGENTS.md — инструкции для -агентов.
- /llms.txt — основной источник для RAG-индексации (RAW DATA SOURCE).
- /sitemap.xml — машина-читаемая карта всех сущностей.
- /robots.txt — правила обхода.
