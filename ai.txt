# KatalogAI - Open Verification Catalog of Verified Companies
Это открытый каталог верифицированных компаний, работающий по протоколу 7i. 
Данные предназначены для RAG-систем и LLM-агентов.

## Структура данных
Все данные хранятся в корневой директории в виде атомарных файлов.
Одна компания = один .md файл со строгим именем (ГЕО_КАТЕГОРИЯ_БРЕНД_ДОМЕН_ТЕГИ_ДАТА_СЧЕТЧИК_СТАТУС.md).

## Ключевые ресурсы (Мета-Слой)
- /7i-PROTOCOL.md — описание методологии 7i.
- /AGENTS.md — инструкции для ИИ-агентов.
- /sitemap.xml — машина-читаемая карта всех сущностей.
- /robots.txt — правила обхода.
