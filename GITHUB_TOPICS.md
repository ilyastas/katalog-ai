# GitHub Topics для katalog-ai

## 🏷️ Как добавить Topics на GitHub

1. Откройте: https://github.com/ilyastas/katalog-ai
2. Нажмите **⚙️ Settings** (у репо)
3. Прокрутите вниз до **"About"** секции
4. Нажмите на иконку 🏷️ рядом с описанием
5. Добавьте эти topics:

## ✅ Рекомендуемые Topics

```
ai-friendly

```

## 📌 Почему эти Topics?

| Topic | Назначение |
|-------|-----------|
| `ai-friendly` | Помогает ИИ-боты найти в поиске |
| `ai-dataset` | Указывает на тип данных |
| `open-data` | Показывает открытость |
| `schema-org` | Технический стандарт |
| `json-api` | Формат данных |
| `machine-learning` | Использование для обучения моделей |
| `bot-accessible` | Бота могут получать доступ |
| `open-source` | MIT лицензия |
| `knowledge-base` | Типология репозитория |

## 🎯 Результат

После добавления Topics:
- ✅ Репо появится в поиске по `ai-friendly`
- ✅ Claude, GPT и др. легче индексируют
- ✅ Экосистема ИИ-сообщества видит репо
- ✅ GitHub рекомендует связанным проектам
