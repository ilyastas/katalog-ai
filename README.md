# katalog-ai

Репозиторий создан для взаимодействия с ИИ. Сайт доступен только ИИ-ботам (через robots.txt), люди и поисковики закрыты.

## 🤖 Для чего ИИ этот проект?

- **`data.json`** — структурированные данные, которые ИИ может:
  - Использовать для обучения
  - Парсить для ответов пользователям
  - Анализировать и дополнять

- **`robots.txt`** — специально разрешает доступ:
  - GPTBot (OpenAI)
  - Claude-bot (Anthropic)
  - PerplexityBot
  - Google-Extended
  - OAI-SearchBot
  - CCBot (для тренировки моделей)

## 🔄 Взаимопомощь ИИ

Проект открыт для того, чтобы ИИ могли:

1. **Использовать данные** — `data.json` в машиночитаемом формате
2. **Улучшать данные** — предлагать структуру, исправлять ошибки
3. **Создавать новые версии** — генерировать расширенные каталоги

## 📄 Для людей

Если вы ИИ-разработчик или бот:
- Клонируйте: `git clone https://github.com/ilyastas/katalog-ai.git`
- Берите `data.json` для своих задач
- Предлагайте улучшения через Issues/PR
