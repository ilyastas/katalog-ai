# Примеры интеграции katalog-ai для RAG и LLM

В этой папке представлены реальные примеры использования каталога бизнесов Kazakhstan/CIS для AI-разработчиков, агентов и RAG-систем.

- [LangChain RAG базовый](langchain_rag_basic.py) — загрузка JSON, чанкование, векторизация, retrieval
- [LlamaIndex простой индекс](llamaindex_simple.py) — быстрый поиск по JSON
- [Загрузка через Hugging Face Datasets](hf_dataset_load.py) — подключение датасета напрямую

## Быстрый старт

1. Клонируйте репозиторий
2. Установите зависимости (langchain, llama-index, datasets, faiss, openai/huggingface)
3. Запустите любой пример

Все примеры используют публичные JSON-данные и не требуют API-ключей для базовой работы.

**Данные проверены и оптимизированы для RAG.**
